# Áudios da Narradora

Coloque os arquivos `.mp3` nesta pasta.
O código carrega via: `./sound/voice/<nome>.mp3`

## Arquivos esperados pelo jogo

| Arquivo | Quando toca |
|---------|-------------|
| `match_start.mp3` | Início da partida |
| `electric_walls.mp3` | "Em 30 segundos paredes elétricas surgiram" |
| `double_kill.mp3` | Double kill |
| `triple_kill.mp3` | Triple kill |
| `killing_spree.mp3` | Sequência de kills |
| `first_blood.mp3` | Primeira morte da partida |
| `enemy_wave.mp3` | Nova onda de inimigos |
| `boss_incoming.mp3` | Chefe se aproximando |
| `low_hp.mp3` | HP crítico do jogador |
| `item_legendary.mp3` | Item lendário coletado |
| `victory.mp3` | Vitória |
| `defeat.mp3` | Derrota |
| `level_up.mp3` | Subiu de nível |
| `time_warning.mp3` | "Faltam 60 segundos" |
